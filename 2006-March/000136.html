Just do it manually once, create a macro and put it into a folder-hook:

   1) uncollapse all
   2) untag all
   3) tag all new
   4) collapse all
   5) tag-exec: uncollapse thread

It works manually so it can be automated. :-)

Still it's not really good for larger threads.
