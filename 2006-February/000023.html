Sorry but no. There's not much logic behind the snapshots as they're 
simply archives from the latest devel tree (and not only produced when a 
change happened). A very poor man's version could be to go by file size.

When changes are made, a mail listing the changes is sent to the 
developer's list but determing whether something changed within a week 
or so requires manual interaction and changelog reading.

Or you could try the subversion interface at our berlios site.
