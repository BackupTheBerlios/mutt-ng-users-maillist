On Wed, May 17, 2006 at 10:23:43AM +0200, Christian Anton wrote:
>Hi folks,
>
>i am using muttng since a few days, have read thousands of howtos
>and manpages to configure muttng/vim to have all the nice features
>like i want them - great!
>
>But what i did not find a solution for is that when a message is
>sent to a mailinglist i subscribed in the index there does not
>appear the sender of the message but something like "To
><mailinglist>". As i have serverside filtering for mailinglist
>messages into some folders i do know that this message belongs to
>the list, but want to see in the index who sent the message.
>
>I hope you understand my bad english...
>
>Thank you in advance for any hints.

Take a look at index_format. The default contains %L:

	%L If an address in the To or CC  header  field
	   matches  an address  defined  by  the users “subscribe”
	   command, this displays ”To <list-name>”, otherwise the same
	   as %F.

You might want to change it to using %F instead:

	%F author name, or recipient name if the message is from you
